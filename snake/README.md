Snake (Tkinter)

Prérequis
- Python 3 (Tkinter est inclus sur Windows/macOS par défaut).

Lancer
- Windows/macOS/Linux:
  - `python snake/main.py`

Contrôles
- Flèches: diriger le serpent
- P: pause/reprendre
- R: recommencer après Game Over

Notes
- Aucune dépendance externe (pas de pygame nécessaire).
